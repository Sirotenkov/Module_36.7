# Module_36.7
Приложение представляет из себя **многопользовательский чат с графическим пользовательским интерфейсом**.

Возможные режимы работы в программе:
1) в качестве пользователя
2) в качестве администратора

Каждый пользователь должен иметь идентификационные данные - уникальный логин и пароль.
Данные для входа в систему от имени администратора жёстко заданы и их изменение не предусмотрено:
1) логин - admin
2) пароль - 1234

Для работы прогораммы необходимо создать базу данных для хранения информации с тремя таблицами:
1) users - для хранения данных о пользователях (логин и пароль для входа в систему)
2) message - для хранения сообщений
3) blacklist - для хранения логинов пользователей, внесённых в чёрный список
Скрипт для создания базы данных находится в папке "scripts".

Пользователи, внесённые администратором в чёрный список, имеют возможность войти в систему под своими учётными данными, но не имеют возможности отправлять сообщения (кнопки отправки деактивированы).
